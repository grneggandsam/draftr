# Drafter

The goal is to make a 2d drafting program. Stretch goal is to make it be able to animate.

Basic Functionality:
- X draw line from point A to point B
- X "snaps"
- X movable
- X zoomable
- Buttons
- "hotkey distances"
- X "hotkey directions"
- selectable lines
- selectable points
- save files
- read files

Advanced
- Annotate
- Layers
- Curves
- Animations
